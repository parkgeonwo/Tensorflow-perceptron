# Tensorflow-perceptron
텐써플로우 2.0을 활용하여 퍼셉트론 구현하기

▩ 퍼셉트론 (perceptron)? 

- 인간의 뇌세포 하나를 컴퓨터로 흉내낸것
- 1957년에 프랑크 로젠블라트가 퍼셉트론을 고안했다.
- 사람의 뇌의 동작을 전기 스위치 on/off 로 흉내낼 수 있다는 이론을 증명
